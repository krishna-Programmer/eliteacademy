Student Exam Manager User Guide — Elite Medical Academy, Khammam

Everything you need to manage exam marks, auto-calculate rankings, and share results across all your devices. No technical knowledge required.

☁ Auto Cloud Sync 📱 Phone & Tablet Ready 🏆 Instant Rankings 📄 Load from Google Sheets 📊 Export to Excel 💾 Backup & Restore

In simple words — what does this do?

Think of this as a smart digital mark sheet. You open a webpage, enter student marks for Botany, Zoology, Physics, and Chemistry, and it automatically calculates each student's total and ranks them — subject-wise and overall.

Every change you make is saved immediately and sent to the cloud, so whether a colleague opens it on their office computer or you check it on your phone at home — everyone sees the same up-to-date data instantly.

There is nothing to install. No login is needed. Just open the link in any browser and start working.

ℹ️

No technical knowledge needed. This guide uses plain, everyday language. If you can use WhatsApp or browse a website, you can use this app.

📁 Managing Exams

You can store many different exams in one place — CT-1, CT-2, Longterm, Half Yearly — each completely separate. Think of each exam as its own separate page in a register book.

Switching between exams

At the top left of the page there is a dropdown box showing the name of the exam you are currently looking at. Tap or click it to see a list of all your exams. Select any one — the mark sheet changes instantly.

Creating a new exam

  • 1
    Click + New

    A small pop-up box appears asking you to type a name.

  • 2
    Type the exam name and click OK

    Use a clear name like CT-2 March 2025 or Longterm CT-5. The new, empty exam opens right away.

Deleting an exam

Select the exam from the dropdown, then click Delete. A warning message will appear asking you to confirm. Click OK only if you are absolutely sure — this removes all student data in that exam permanently.

🚨

Always take a Backup before deleting anything. See the Backup section below. It takes about 5 seconds and means you can always recover the data if you change your mind.

👤 Adding Students

ℹ️

Every student must have a unique Student ID. This ID is like a permanent roll number — it's how the app recognises each student and keeps their data correct across all updates. Use your academy's existing roll number or admission number (for example: EMA-001). The student's name and marks are optional, but the ID is always required.

There are two ways to add students: one by one (good for a handful of students), or all at once by uploading a Google Sheet (recommended when you have many students — see the next section).

Adding one student at a time

  • 1
    Switch to Edit Mode

    Click the ✏ Edit Mode button at the top. The mark sheet now shows small typing boxes in each cell.

  • 2
    Click + Student

    A pop-up asks for the Student ID. Type the roll number (e.g. EMA-042) and click OK.

  • 3
    Type the name and marks

    A new row appears at the bottom. Click the Name cell and type the student's name. Then click each subject cell and type the mark.

  • 4
    Total and ranks update by themselves

    The moment you finish typing a mark and click elsewhere (or press Tab on a keyboard), the Total and all Rank columns recalculate automatically. Nothing else to do — it's all saved too.

Removing a student

In Edit Mode, each student row has a small ✕ button at the far right end of the row. Tap it to remove that student. All other students' ranks will update automatically.

⚠️

If you try to add a student with an ID that already exists in this exam, the app will warn you and not add a duplicate. Each student must have a different, unique ID.

📄 Loading Students from a Google Sheet

Instead of typing students one by one, you can prepare a list in Google Sheets (or Microsoft Excel) and load it into the app all at once. This is the recommended method for large batches.

ℹ️

Smart update — never worry about loading twice: If you load a sheet and a student's ID already exists in the app, only the columns present in that sheet are updated — nothing else is touched. New IDs are added as new students. Rows with no ID are skipped. You can safely load the same sheet multiple times.

Step 1 — Prepare your Google Sheet

  • 1
    Open Google Sheets on any device

    Go to sheets.google.com and create a new sheet, or open an existing one.

  • 2
    Type the column headings in the first row

    In the very first row, type the following words — one per cell — starting from cell A1:
    id   name   bot   zoo   phy   che

    You don't need all of them. The only one that is required is id. Only add the columns you actually have data for.

  • 3
    Fill in student data from row 2 onwards

    One student per row. The ID column must always have a value. Other columns can be left empty.

  • 4
    Download the sheet as a CSV file

    In Google Sheets, click File at the top → Download → Comma-separated values (.csv). A file will download to your device's Downloads folder.

Step 2 — Load the file into the app

  • 1
    Click Load CSV in the toolbar

    A file browser opens — exactly like when you attach a photo on WhatsApp.

  • 2
    Find and select the file you just downloaded

    On Android: it's in your Downloads folder. On iPhone: tap Files → Downloads. On a computer: check your Downloads folder.

  • 3
    Check the result message at the top right

    A message appears, for example: 12 added · 3 updated · 0 skipped ✓ — this tells you exactly what happened: how many new students were added, how many existing ones were updated, and how many rows were skipped (usually because they had no ID).

Example: Registering students at the start of a new exam (names only)

id,name EMA-001,Arjun Reddy EMA-002,Priya Sharma EMA-003,Kiran Babu

Example: Loading marks after the exam (using the same IDs)

The app finds each student by their ID and fills in only their marks. Names and other details stay unchanged.

id,bot,zoo,phy,che EMA-001,85,92,78,88 EMA-002,91,87,95,89 EMA-003,76,80,70,74

Example: Correcting just one subject's marks

Load a sheet with only the ID and the subject you need to fix. Everything else stays exactly as it was.

id,bot EMA-001,90 ← only Botany mark is changed for this student
💡

The order of columns in your sheet doesn't matter. The heading words don't need to be exactly right either — the app also accepts botany, zoology, physics, chemistry and even capital letters like BOT or Bot.

✏️ Entering and Correcting Marks

You can type marks directly into the mark sheet while in Edit Mode. Every change is saved automatically the moment you move to the next cell — there is no Save button needed.

How to enter marks

  • 1
    Click ✏ Edit Mode

    This button is in the toolbar at the top. Once you click it, the mark sheet shows small typing boxes in each mark cell.

  • 2
    Click on a cell and type the mark

    Tap or click any mark cell. Type the number. On a keyboard, press Tab to jump to the next cell, or Enter to move down.

  • 3
    The total and ranks update instantly

    The moment you move away from a cell, the Total and Rank columns recalculate automatically. Your change is saved to the cloud at the same time — you will see a brief "Syncing…" then "Synced ✓" message.

Correcting a mark that was entered wrong

In Edit Mode, simply click the wrong mark, delete it, type the correct number, and click elsewhere. The total and ranks will fix themselves immediately. This works at any time — even months after the marks were first entered.

ℹ️

You cannot type in the Total column — it is always auto-calculated as Botany + Zoology + Physics + Chemistry. Only the four subject marks need to be entered.

⚠️

Try not to edit on two devices at exactly the same time. If two people are both making changes simultaneously, only the last change to be saved will be kept. Best practice: have one person enter marks at a time, or split subjects (one person enters Botany & Zoology, another enters Physics & Chemistry) and both sync when done.

🏆 Rankings & Sorting

How ranks work

Ranks are calculated automatically the moment any mark changes. You never have to calculate or type rank numbers yourself.

There are five rank columns: one for each subject (Botany, Zoology, Physics, Chemistry) and one for the Overall Total. Each is independent — a student can be Rank 1 in Botany but Rank 5 overall.

Tied marks are handled fairly: If two students both score the highest in a subject, they both get Rank 1, and the next student gets Rank 3 (not Rank 2) — exactly like a competitive exam.

What the colours mean

ColourMeaning
🟡 Yellow / GoldRank 1 — Top scorer in that subject or overall
🔵 Light BlueRank 2
🟢 Light GreenRank 3
No colourRank 4 and below

Sorting the table by rank

To see the top performers at the top of the table, tap or click any rank column heading:

  • Tap B.Rank → sorts by Botany rank (best scorer at top)
  • Tap Z.Rank → sorts by Zoology rank
  • Tap P.Rank → sorts by Physics rank
  • Tap C.Rank → sorts by Chemistry rank
  • Tap Rank (last column) → sorts by Overall Total rank

The sorted column turns blue and shows a ▼ arrow (best at top) or ▲ (reversed). Tap the same column again to flip the order. Tap a different column to sort by that one instead.

Note: sorting only changes how the table looks on screen — it does not change the actual data or the ranks themselves. Everything is still calculated correctly.

Hiding columns to simplify the view

If the table feels too wide on your phone, tap ⚙️ to open the column panel. Untick any columns you don't need right now. For example, hide the individual subject rank columns and keep only the Total Rank column — this makes the table much easier to read on small screens. Hidden columns are also left out of Excel and CSV exports.

☁️ Cloud Sync — How Your Data Stays Safe & Shared

Every time you make a change — adding a student, entering a mark, anything — the app automatically saves that change to a secure cloud server. This is what keeps all devices in sync and protects your data.

What happens automatically (nothing you need to do)

  • When you open the app: It loads data already stored on your device instantly (so the page opens fast), then quietly checks the cloud for any newer data. If newer data is found — for example, a colleague entered marks on another device — it updates your view automatically.
  • Every time you make a change: The change is pushed to the cloud within a few seconds. You will see a brief Syncing… message appear, followed by Synced ✓.

The ☁ Sync button — when should you press it?

In most situations you will never need to press this — sync happens on its own. But press it in these situations:

Situation 1 — You just got your internet back

If you entered marks without internet and the connection just returned, press ☁ Sync to push all your changes to the cloud immediately.

Situation 2 — Before handing over to a colleague

Press ☁ Sync first to make sure your colleague sees your latest data the moment they open the app on their device.

Situation 3 — You see "Sync failed ✗"

This usually means internet was unavailable at that moment. Your data is safe on your device. Once the internet is back, press ☁ Sync to try again.

Understanding the status messages

MessageWhat it meansWhat to do
Syncing… Saving to the cloud right now Wait a moment. Nothing to do.
Synced ✓ Cloud is up to date. All good. Nothing to do.
Sync failed ✗ Could not reach the cloud — usually means no internet at that moment Check your connection. Press ☁ Sync when back online. Data is safe on your device.
⚠️

Avoid editing on two devices at exactly the same moment. If two people save different things at precisely the same second, only one will go through. To avoid this, have one person enter marks at a time. If two people must work simultaneously, split subjects between them and both press ☁ Sync when done.

💾 Backup — Saving a Safety Copy on Your Device

Cloud sync keeps your data safe online, but a Backup is an extra copy saved directly on your phone or computer — like a photocopy of the register that you keep in a drawer. If something ever goes wrong, you can restore from this copy.

🚨

Always take a Backup before: deleting an exam, loading a large file of marks onto existing data, or restoring an old backup. It takes about 5 seconds and can save hours of re-work.

How to take a Backup — step by step

  • 1
    Click Backup in the toolbar at the top

    A file is downloaded to your device immediately. No form, no settings — just one click.

  • 2
    The file is saved to your Downloads folder automatically

    The file is named something like exam-backup-2025-06-15.json with today's date in the name. You don't need to understand what ".json" means — just treat it as your backup file.

    The Backup file contains all your exams, not just the current one — so one backup protects everything.

  • 3
    Move it somewhere safe for easy retrieval

    Send it to yourself on WhatsApp, email it to yourself, or move it to Google Drive. Keep backups from the last few sessions so you can go back in time if needed.

💡

Make backing up a habit. A quick click of Backup at the end of each mark entry session means you always have a recent copy to fall back on. Takes 5 seconds.

🔄 Restore — Getting Your Data Back from a Backup

If something goes wrong — data accidentally deleted, a mistake during a large load, or any other problem — you can bring back your data from a Backup file you saved earlier. This is simple and takes less than a minute.

How to restore from a backup — step by step

  • 1
    Find your backup file

    Locate the backup file you saved earlier — from your Downloads folder, Google Drive, WhatsApp, or email. The file name ends with .json.

  • 2
    Click Restore in the toolbar

    A file browser opens — the same kind that opens when you attach a file on WhatsApp.

  • 3
    Select your backup file

    Navigate to where you saved it and tap or click on the file name to select it.

  • 4
    Choose how to restore

    A question appears with two choices:

    Click OK → Replace everything: Wipes all current data and brings back exactly what was in the backup. Use this when something went badly wrong and you want to go back to a previous state.

    Click Cancel → Add alongside (Merge): Keeps all your current data and also brings in the exams from the backup. If both have an exam with the same name, the backup version takes over. Use this when you want to recover just one or two exams without losing everything else.

  • 5
    Done — data is restored

    The page refreshes and shows the restored data. A green "Backup restored ✓" message confirms it worked. The restored data is also pushed to the cloud automatically — so all other devices will see the restored data the next time they open the app.

ℹ️

Not sure which option to pick? Choose Cancel (Add alongside / Merge) — it's the safer choice because it doesn't delete any of your current data. You can always do a full Replace later if needed.

⚠️

Restoring affects all devices. Because the restored data is pushed to the cloud, every device that opens the app afterwards will see the restored data. Make sure this is what you want before you confirm.

📤 Exporting & Printing

Save the mark sheet as an Excel file, download it as a simple spreadsheet file, or print it as a PDF — for records, sharing with management, or printing for students and parents.

Export to Excel (for records or sharing)

  • 1
    Click Excel

    A file is downloaded to your device straight away. The file name is the exam name (e.g. LT CT-5.xlsx).

  • 2
    Open it in Excel or Google Sheets

    On Android: tap the downloaded file. On iPhone: tap Files → Downloads → the file. On a computer: double-click it in your Downloads folder. It opens in any spreadsheet app.

💡

Only the columns you have visible on screen are included in the Excel file. If you've hidden some columns using ⚙️, those won't appear in the export. You can use this to create a clean, simple export with only the columns you need.

Print as PDF (for physical copies or WhatsApp sharing)

  • 1
    Prepare the view first

    Sort by the Total Rank column (tap Rank heading) so the best students appear first. Use ⚙️ to hide any columns you don't want on paper. Make sure you are in 👁 View Mode (not Edit Mode) so typing boxes don't show up in the printout.

  • 2
    Click PDF

    Your browser's Print window opens. The toolbar buttons disappear — only the clean mark sheet is shown.

  • 3
    Save as PDF or print

    In the print window, select "Save as PDF" to get a PDF file you can share on WhatsApp or email. Or select your printer to print directly. Choose A4 paper size for the best fit.

🗂️ Google Sheet / CSV Format Reference

If you prepare your own sheet to load into the app, here is everything you need to know.

What your sheet must look like

Row 1 = headings. Row 2 onwards = one student per row.

id,name,bot,zoo,phy,che EMA-001,Arjun Reddy,85,92,78,88 EMA-002,Priya Sharma,91,87,95,89 EMA-003,Kiran Babu,76,80,70,74

Accepted column headings

ColumnHeadings the app acceptsRequired?
Student IDid   roll   rollno   student_id✅ Yes — must not be blank
Student Namename   student_nameOptional
Botany marksbot   botanyOptional
Zoology markszoo   zoologyOptional
Physics marksphy   physicsOptional
Chemistry marksche   chem   chemistryOptional

Simple rules

  • Every student's ID must be filled in. Rows with no ID are skipped.
  • Column order doesn't matter. The app reads them by name, not position.
  • Capital or small letters don't matter: BOT, Bot, bot are all accepted.
  • If a mark column is empty for a student, it defaults to 0.
  • Do not add Total or Rank columns — the app calculates these itself.
  • Extra columns not in the list above (e.g. attendance) are silently ignored.

💡 Tips & Tricks

📋 Recommended workflow for every new exam

  1. Click + New and give the exam a clear name (e.g. CT-2 March 2025)
  2. Load student names and IDs from Google Sheets using Load CSV
  3. After the exam, enter marks into the same Google Sheet and load it again
  4. Review rankings — sort by the Rank column to see top performers
  5. Export to Excel for records, or print as PDF for notice boards and parents
  6. Click Backup to save a safety copy to your device

📊 Reuse the same student list for every exam

At the start of the academic year, create your student list (ID + Name) in Google Sheets once. For every new exam, load that same sheet — students are added in seconds. Then load marks separately when ready. You never need to retype names.

📱 Fastest way to enter marks on a phone

Tapping small cells in a table on a phone can be fiddly. Instead: fill the marks into Google Sheets on your phone, download as CSV, then load into the app. This is much faster than cell-by-cell typing on a small screen.

🖨️ Getting a clean printout

  1. Sort by Total Rank (tap the Rank heading) — best students at the top
  2. Use ⚙️ to hide any columns not needed on paper
  3. Switch to 👁 View Mode so typing boxes disappear
  4. Click PDF → choose "Save as PDF" → A4 size

🔁 Just found a wrong mark?

Switch to Edit Mode, find the student (you can sort by name using the ID column), click the wrong mark, type the correct number, and click elsewhere. Ranks fix themselves instantly and the correction is saved to the cloud. Nothing else is needed.

🚨

The golden rule: Before any big change — deleting an exam, loading a large file onto existing data, restoring a backup — always click Backup first. Five seconds of effort can save hours of re-work.
